# FUTURE_DS_01
This project focuses on analyzing e-commerce sales data from a Superstore dataset to uncover meaningful business insights.
An interactive Power BI dashboard was created to help business owners and decision-makers understand sales performance, customer behavior, and profitability.
The dashboard answers key business questions such as:
1)What are the best-selling products?
2)When do sales peak during the year?
3)Which categories and regions generate the most revenue?
